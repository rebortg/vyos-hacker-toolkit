==========
VyoS-Extra
==========

Introduction
============

A tool to build VyOS packages, ISO, and more.

Installation
============

Prerequisites
-------------

It requires a recent python 3 version (3.7 or later recommended). It includes/vendors its dependencies.

Using pip
---------

#. Use pip to install the packages:

::

    pip install -U vyos-extra
    vyos --help
